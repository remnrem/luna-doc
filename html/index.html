Luna: software for the analysis of sleep signal data

Open-source C/C++ tools with Python and R bindings for sleep EEG and PSG analysis.

Getting started Main concepts Luna commands Walk-through

lunascope

Desktop viewer app that incorporates Luna.

lunaC

Command-line workflows for large datasets and scripted analyses.

lunapi

Python interface for interactive work and notebooks.

lunaR

For users who prefer the R ecosystem.

Highlights

  • Hypnogram-based macro-architecture metrics and automated staging
  • Manipulate signal (EDF) and annotation (event-level) data
  • Spectral analyses, spindles and slow oscillations
  • Artifact detection and epoch masks
  • Connectivity and interval-based analyses
